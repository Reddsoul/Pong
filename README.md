Player 1 (Left Paddle):
              W - Move Up
              S - Move Down
              A - Move Left
              D - Move Right
              
Player 2 (Right Raddle):
              I - Move Up
              K - Move Down
              J - Move Left
              L - Move Right
            
Other Controls:
              SPACE - Start/Resume Game
              R - Reset Game
              ESC - Quit Game
